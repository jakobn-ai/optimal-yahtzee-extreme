# optimal-yahtzee-extreme

Play optimal solitaire Yahtzee according to ["An Optimal Strategy for Yahtzee"](http://gunpowder.cs.loyola.edu/~jglenn/research/optimal_yahtzee.pdf), Glenn 2006; including support for the d10 "Kniffel Extreme" variant, which I believe is only sold in German-speaking countries? Currently work in progress.
